# Term Project: Stardew Valley Database Manager

โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา ENGSE219 Database Systems โดยมีวัตถุประสงค์เพื่อออกแบบและพัฒนาระบบฐานข้อมูลสำหรับเกม Stardew Valley โปรเจกต์ประกอบด้วยการออกแบบฐานข้อมูลเชิงสัมพันธ์, การสร้างสคริปต์ SQL, และการพัฒนาเว็บแอปพลิเคชันสำหรับจัดการข้อมูล

✨
## Features

* **จัดการข้อมูลชาวสวน (Farmer):**
    * แสดงข้อมูล chi tiết ของชาวสวนแต่ละคน เช่น ระดับทักษะ, เงิน, และค่า Buffs รวมจากอุปกรณ์
* **ระบบจัดการช่องเก็บของ (Inventory):**
    * เพิ่มไอเทมใหม่เข้าสู่ช่องเก็บของของชาวสวนได้ผ่านฟอร์มบนหน้าเว็บ
* **แสดงผลอันดับ (Leaderboard):**
    * แสดงอันดับชาวสวนทั้งหมด โดยเรียงลำดับจากผู้ที่มีเงินเยอะที่สุด
* **ระบบ Query แบบไดนามิก:**
    * เรียกดูข้อมูลที่ซับซ้อนผ่าน UI
    * แสดงผลลัพธ์ในรูปแบบตารางที่อ่านง่าย
